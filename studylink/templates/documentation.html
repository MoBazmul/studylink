1. Introduction

StudyLink is a resource-sharing platform designed for tech students to collaborate, share study materials, connect with peers, and integrate social media and GitHub profiles, fostering a collaborative learning community.

2. Getting Started

2.1 Installation

git clone https://github.com/yourusername/studylink.git
cd studylink
pip install -r requirements.txt

Create a .env file with environment variables:

SECRET_KEY=your_secret_key
DATABASE_URI=your_database_uri
YOUTUBE_API_KEY=your_youtube_api_key

2.2 Usage

python app.py

Access StudyLink in your web browser at http://localhost:5000.
